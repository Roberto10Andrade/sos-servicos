# SOS Serviços - Plataforma de Prestadores de Serviços

Uma plataforma web moderna para conectar prestadores de serviços e clientes.

## 🚀 Funcionalidades

- 🔍 Busca de profissionais por categoria
- 👤 Cadastro de prestadores de serviços
- ⭐ Sistema de avaliações
- 💬 Contato direto via WhatsApp ou email
- 📱 Design responsivo

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3 (com Flexbox e Grid)
- JavaScript (ES6+)
- LocalStorage para persistência de dados
- Font Awesome para ícones

## 📱 Responsividade

O site é totalmente responsivo e funciona em:
- 💻 Desktops
- 📱 Tablets
- 📱 Smartphones

## 🎨 Design

- Interface moderna e intuitiva
- Animações suaves
- Efeitos visuais modernos (glassmorphism)
- Paleta de cores profissional

## 🚀 Como Usar

1. Clone o repositório
2. Abra o arquivo `index.html` em seu navegador
3. Explore as funcionalidades!
